# CRUD based API for basic authentication

```
Here's what I have done!

    I have used:
    - Restify nodejs framework   --for creating REST API
    - Mongoose   --for mongoDB Database Connectivity
    - Dotenv  --for Environmental accessing Variables
    - nodemon  --for only DEV environment
    - mongodb-timestamp --to document time while adding some data in database
    - restify-errors --for handling errors along the way
```


### Getting started guide!
 
```diff
Steps to setup it localy if needed!
    1. Fetch the repo localy
    2. install node_modules by "npm i" in your terminal
    3. add your MongoDB_URI String in the ENV Variable file.
    4. "npm start" start server in Production ENV
    5. "npm run dev" start server in Dev ENV 


##  CRUD FUNCTIONALITY ##

Functionality           Info

    Create      - create users in database
    Read        - read users in database  
    Update      - update users in database
    Delete      - delete users in database

```
